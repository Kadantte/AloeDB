# REST Api
Simple REST Api example, made with [AloeDB](https://github.com/Kirlovon/AloeDB), [Oak](https://github.com/oakserver/oak), [Superstruct](https://github.com/ianstormtaylor/superstruct) and [Nano ID](https://github.com/ai/nanoid).

## Start
```
deno run --allow-read --allow-write --allow-net .\server.ts
```
